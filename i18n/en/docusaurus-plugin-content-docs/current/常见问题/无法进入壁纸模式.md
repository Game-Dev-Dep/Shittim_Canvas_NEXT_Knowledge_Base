# Can’t enter wallpaper mode? {#cannot-enter-wallpaper}

Wallpaper mode won’t stick to the bottom—or crashes? Try triage in order.

***

1. Quit other wallpaper apps (Wallpaper Engine, Lively, etc.).
2. Pause desktop “beautifiers” (e.g. CoolDesk, Tencent Desktop Organizer, Rainmeter stacks).
3. Check antivirus logs—your AV might be “helpfully” blocking layering tricks.
4. Restart **Windows Explorer** from Task Manager.
5. Reboot—Window Manager in a good mood fixes more than we’d like to admit.

:::warning

If none of that helps, reach out on any [community channel](../关于/联系我们.md) and attach your [log file](./如何查找日志文件.md#log-file).

:::
