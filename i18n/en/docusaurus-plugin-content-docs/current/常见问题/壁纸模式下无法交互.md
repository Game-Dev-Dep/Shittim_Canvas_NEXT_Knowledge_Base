# Can’t interact in wallpaper mode? {#wallpaper-interaction}

If wallpaper clicks feel “dead,” something is probably sitting on top.

***

If the [window overlay indicator](../功能总览/壁纸模式.md#wallpaper-mode) shows up, a window is blocking wallpaper input. Add it to the allowlist with any of the filter rules—pick your poison.
