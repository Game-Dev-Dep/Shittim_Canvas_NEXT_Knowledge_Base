# Why isn’t the SCNX repo open like SC’s? {#why-not-open-source}

The short story behind the curtain.

***

SCNX leans on some **creative implementation details** to match game behavior. Open-sourcing that would be… *loud*. Think “attention we don’t need.”

Want to contribute anyway? Talk to the [developers](../intro.md#developers). Genuine help is genuinely welcome.

Worried about malware backdoors? Also talk to us. “Closed source” is mostly **risk management**, not a secret villain arc. No evil lair—promise. Meow.
