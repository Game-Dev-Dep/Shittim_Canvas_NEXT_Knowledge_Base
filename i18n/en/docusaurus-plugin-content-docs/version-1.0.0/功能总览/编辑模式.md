# Edit mode {#edit-mode}

![](./img/09.png)  

### Shortcuts
- On the **main view**, press **`Ctrl+E`** to enter edit mode.
- In **edit mode**, **`Ctrl+E`** exits.
- In **edit mode**, **`Ctrl+R`** resets the character transform (when you’ve… experimented too hard).

### Mouse
- **Edit mode** + **left drag** — move the character.
- **Edit mode** + **middle scroll** — rotate.
- **Edit mode** + **right drag** — scale.
