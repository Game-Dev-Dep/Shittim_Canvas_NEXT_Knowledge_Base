# Can’t click the window overlay indicator? {#kivotos-finder}

The **KIVOTOS FINDER** bar ignores you? Same—here’s a workaround buffet.

***

Root cause: still TBD. Try adding the offending window to the **ignore** list manually—**one** of these is enough:

1. Open `Shittim Canvas NEXT\Core Files\Config Files\Custom_Window_Config.json` in any editor.
2. **`Custom_Exclusive_Process_Name_List`** — process **display names**, e.g. `"Custom_Exclusive_Process_Name_List": ["Settings", "Calculator"],`
3. **`Custom_Exclusive_Handle_List`** — **window handles**, e.g. `"Custom_Exclusive_Handle_List": ["00000001", "00000002"],` (handles change—use sparingly).
4. **`Custom_Exclusive_Class_Name_List`** — **window class** names, e.g. `"Custom_Exclusive_Class_Name_List": ["Chrome_WidgetWin_1", "Chrome_WidgetWin_2"],`
5. Restart the app.

:::
