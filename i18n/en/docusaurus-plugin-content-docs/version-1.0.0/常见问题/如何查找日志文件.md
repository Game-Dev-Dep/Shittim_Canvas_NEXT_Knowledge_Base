# Where is the log file? {#log-file}

When support asks for logs, this is the treasure map.

***

`Player.log` lives at:

`C:\Users\<YourUsername>\AppData\LocalLow\Game Dev Dep\Shittim Canvas NEXT\Player.log`

**Main menu** → **Settings** → **About** → **Log folder** opens that folder in one click—no spelunking required.
