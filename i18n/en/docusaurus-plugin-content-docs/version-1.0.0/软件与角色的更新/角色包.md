# Character pack {#character-pack}

Characters only—grab when you already have the app.

Archives like **`[角色] SCNX V(x.x.x) (xxx).zip`** contain **character assets and metadata** only.

![](./img/03.png)

Extract **on top of** your install folder—same level as **`Shittim Canvas NEXT.exe`**—and let files merge/overwrite when prompted. Backup first if you’re attached to your layout.
