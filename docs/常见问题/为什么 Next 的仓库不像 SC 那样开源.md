# SCNX 的仓库为什么不像 SC 一样保持开源？

了解一切事情背后的缘由.jpg

***

由于 SCNX 的项目大量使用了一些神秘原理来进一步复刻游戏的各种效果与逻辑，因而不适合开源，容易招来神秘大手。

如果您有意向向 SCNX 贡献代码的话，欢迎与[开发者](../intro.md#开发者)交流，只要是真心想为软件的开发出一份力的话，我们无比欢迎！

如果您担心 SCNX 存在恶意代码与后门的话，也欢迎与[开发者](../intro.md#开发者)交流，不开源只是防止被橄榄，而不是开发者真的精力过剩，有什么不可告人的邪恶阴谋，敬请谅解喵。
