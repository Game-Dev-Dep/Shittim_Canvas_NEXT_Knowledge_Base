# 无法进入壁纸模式？

查看壁纸模式无法沉底或者崩溃等问题的解决方法

***

该问题的成因有很多，我们建议逐步排查：  

1) 关闭其他壁纸软件（如 Wallpaper Engine、Lively 等）。
2) 关闭桌面美化工具（如 CoolDesk、腾讯桌面整理、雨滴 等）。
3) 关闭或查看杀毒软件日志，确认操作是否被拦截。
4) 在任务管理器中重启资源管理器。
5) 重启计算机以保证窗口管理器处于正常状态。

:::warning

如上述操作无法解决问题，请在任意[反馈渠道](../关于/联系我们.md)向开发者提供[日志文件](./如何查找日志文件.md)

:::
