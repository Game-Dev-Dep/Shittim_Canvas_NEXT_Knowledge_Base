# 如何在音乐播放器中添加本地音乐？

了解如何在音乐播放器中添加自己喜欢的音乐

***

1) 使用任意文本编辑器打开 `Shittim Canvas NEXT\Core Files\Config Files\BGM_Meta_Info_Config.json`  
2) 参考已有自定义歌曲条目 "-1" 来修改，（注意：自定义歌曲的 ID 应为 **负数**，所以下一个自定义元数据的 ID 应该为 -2），目前支持 `mp3`、`flac`、`aiff`、`wav`、`ogg` 格式
``` json
"-1": {
    "BGM_Id": -1,
    "BGM_Name": "TV WORLD",
    "BGM_Author": "Toby Fox",
    "BGM_File_Path": "D:/Games/Steam/steamapps/common/DELTARUNE/mus/tv_world.ogg"
}
```
3) 重启应用
